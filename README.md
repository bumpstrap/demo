# Bumpstrap Demo Template

A comprehensive Bootstrap 4 template used for testing [Bumpstrap](https://github.com/bumpstrap) — the automated Bootstrap upgrade service.

---

## Contents

29 HTML pages covering virtually every Bootstrap 4 component:

- **Layout:** Media Object
- **Content:** Typography, Code, Images, Tables, Figures
- **Components:** Alerts, Badge, Breadcrumb, Buttons, Button Group, Card, Carousel, Collapse, Dropdowns, Forms, Input Group, Jumbotron, List Group, Modal, Navbar, Navs, Pagination, Popovers, Progress, Tooltips
- **Utilities:** Utilities overview
